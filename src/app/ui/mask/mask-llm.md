# Mask

```typescript
import { Mask } from '@/ui/mask';
```

## Components

### `meeMask`

- **Inputs**
  - `meeMask`: string - The mask pattern to apply to the input
  - `showMaskType`: boolean = false - Whether to show the mask type

## Usage

```html
<mee-mask [meeMask]="'(000) 000-0000'" [showMaskType]="showMaskType" />
```
