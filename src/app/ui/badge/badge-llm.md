# MEE Badge

```typescript
import { Badge } from '@/ui/badge';
```

## Components

### `<mee-badge>, [meeBadge]`

- **Inputs:**

  - `variant`: 'default' | 'secondary' | 'outline' | 'destructive' = 'default' - Badge variant

- **Export:** `meeBadge`

## Usage

```html
<mee-badge>Badge</mee-badge> <button meeBadge variant="secondary">Badge</button>
```
