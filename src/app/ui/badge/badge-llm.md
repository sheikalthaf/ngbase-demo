# MEE Badge

```typescript
import { Badge } from '@/ui/badge';
```

## Components

### `<mee-badge>, [meeBadge]`

- **Inputs:**

  - `variant`: 'info' | 'success' | 'warning' | 'danger' = 'info' - Badge variant

- **Export:** `meeBadge`

## Usage

```html
<mee-badge>Badge</mee-badge> <button meeBadge variant="success">Badge</button>
```
